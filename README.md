# 🏥 Hospital Information System (HIS)

A comprehensive Hospital Information System (HIS) designed to manage hospital data, patient records, and medical processes. This project is developed in two phases, covering everything from core operational management to advanced IoT-based monitoring.

---

## 🚀 Features

### Phase 1: Core System
* **👤 Patient Management:** Detailed demographic tracking, medical history, and vital signs management.
* **📅 Admissions & Scheduling:** Streamlined in-person and online appointment booking.
* **🛏️ Resource Management:** Efficient bed and room assignment (ICU, Emergency, etc.).
* **🧪 Lab & Imaging:** Integrated recording of laboratory results and critical value alerts.
* **💊 Pharmacy & Inventory:** Electronic prescription handling, drug interaction checking, and warehouse stock management.
* **💰 Financials:** Automated billing, insurance calculation, and payment tracking.

### Phase 2: Advanced Features
* **📈 Smart Analytics:** Physician recommendations and bed occupancy analysis.
* **📡 IoT Integration:** Real-time sensor data logging (Heart Rate, SpO2, Temp) and automated alert management.
